Windows Installation Guide

Step 1
Prepare Bootable USB

Download Windows ISO and create a bootable USB using Rufus or Multiple Bootable.

Step 2
Insert USB & Restart

Plug USB, restart PC/Laptop, and press boot key (F2, F11, F12, ESC, DEL).

Step 3
Enter BIOS

Set USB as first boot device and save changes. Disable Secure Boot if necessary.

Step 4
Start Setup

Select language, time, keyboard, then click Install Now.

Step 5
Product Key

Enter your key or skip to activate later.

Step 6
Choose Installation Type

Select "Custom: Install Windows only".

Step 7
Partition Drive

Delete old partitions and select unallocated space.

Step 8
Installing Windows

Wait while files are copied and system restarts. This may take some time.

Step 9
Setup Account

Create username, password, and configure settings.

Step 10
Install Drivers

Update drivers and run Windows Update.

Step 11. Optional
Disk Partitioning

If not using the entire disk, use Disk Management to create, delete, or resize partitions. And configure the partitions as needed.
